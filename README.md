# Simple Todo app
Build using vanilla-JS as a part of personal practice path

# Tech Stack
| Name | Purpose |
| ---- | ---- |
| NodeJS | Server |
| MongoDB | DB |

# Run

To run the server locally. We need to set up db server using Docker. 

```
bash
docker compose up
```

```
bash
npm install
```

```
bash
npm run watch
```
